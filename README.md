# Proyecto React
Proyecto de prueba para realizar proyectos con react y ver el funcionamiento de este. 

Se uso Vite el cual ayuda a optimizar los proyectos en el navegador.

Se uso node para manejar las rutas de navegación dentro del app

## Versiones necesarias para instalación
- Node 18.20.5

## Correr app
- Clonar repositorio

  `git clone `

- Se ingresa a la ruta del proyecto recien clonado

  `cd rutaDelProyecto`

- Se instalan los paquetes npm necesarios

  `npm i`

- Se inicia el app

  `npm run dev `
